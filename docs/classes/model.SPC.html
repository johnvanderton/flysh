Class SPC

'SPC' class model

Handles the 'Scope', 'Parent' and 'Children' DOM objects. This element is direclty involved with the 'Filter Selector' functionality.

Contains : - A scope/parent filter selector signature - Children 'Sibling' elements - Fields record order
