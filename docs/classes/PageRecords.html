    Class PageRecords

    'PageRecords' class model

    The class is instanciated and filled by the 'Flysh' class during page(s) parsing. The data collection is structured as an array of 'Map'. In case of exception during process, the 'error' flag is set to true (i.e: timeout, crash,...).

    Example of the output data format sent back after page(s) parsing, [ ['item_name' : 'some name', 'item_description' : 'some description',...], ['item_name' : 'some name', 'item_description' : 'some description',...], ... ]
